개인정보 처리방침

HeySolver · 최종 업데이트: 2026년 2월 12일

소개

HeySolver는 AI 기반 학습 도우미로, 학생들이 단계별 설명과 안내 질문, 심화 대화를 통해 문제를 이해할 수 있도록 도와줍니다. 본 방침은 당사가 귀하의 정보를 수집, 사용 및 보호하는 방법을 설명합니다.

개인정보 보호 약속

HeySolver는 "데이터 최소화" 원칙을 따릅니다. 핵심 기능 제공에 필요한 데이터만 처리하며, 귀하의 개인정보를 엄격히 보호합니다.

수집 및 처리하는 데이터

핵심 기능을 제공하기 위해 다음 데이터를 처리합니다:

  • 계정 정보: Apple 로그인 또는 Google 로그인을 통해 이메일 주소(공유를 선택한 경우)를 수집하며, 이는 기기 간 계정 동기화를 위해서만 사용됩니다.
  • 문제 이미지: 촬영한 문제 사진은 문제 인식 및 분석에 사용됩니다. 원본 이미지는 서버에 저장되지 않습니다.
  • 문제 텍스트: 입력한 텍스트는 학습 가이드 생성에 사용됩니다. 원본 텍스트는 서버에 저장되지 않습니다.
위치, 연락처 또는 기타 민감한 개인정보는 수집하지 않습니다.

정보 사용 방법

  • AI 기반 학습 설명 및 학습 안내 제공
  • 학습 기록 및 즐겨찾기를 기기에 로컬 저장
  • 데이터 동기화를 위한 계정 인증

제3자 서비스 및 데이터 공유

핵심 기능을 제공하기 위해 다음 제3자 서비스를 사용합니다:

  • AI 서비스 (Google Gemini):
    • 제공자: Google
    • 전송 데이터: 제출하신 문제 이미지 및 텍스트
    • 목적: 문제 인식, 학습 가이드 생성, 심화 대화
    • 데이터 보호: 모든 전송은 HTTPS 암호화를 사용합니다. 데이터는 실시간 처리에만 사용되며 광고나 모델 학습에 사용되지 않습니다 (Google AI 이용약관 참조)
  • 인증 (Supabase):
    • 목적: 안전한 사용자 인증 및 계정 관리
    • 필수 계정 정보만 저장됩니다
  • iCloud (Apple):
    • 목적: 기기 간 데이터 동기화 (선택 사항)
    • 데이터는 귀하의 비공개 iCloud 컨테이너에 저장됩니다
    • 앱 설정에서 언제든지 iCloud 동기화를 비활성화할 수 있습니다
  • 결제 (Apple):
    • 모든 구독 및 구매 거래는 Apple에서 처리합니다
    • 당사는 귀하의 금융 정보를 수집하거나 접근하지 않습니다

데이터가 공유되는 경우

다음 상황에서만 AI 서비스에 데이터가 전송됩니다:

  • "사진 인식" 사용 시: 문제 사진
  • "문제 입력" 사용 시: 문제 텍스트
  • "심화 대화" 사용 시: 대화 메시지 및 현재 문제 맥락

데이터 보안

  • 전송 보안: 모든 네트워크 통신은 HTTPS 암호화를 사용하여 전송 중 데이터의 기밀성과 무결성을 보장합니다.
  • 저장 보안: 원본 이미지와 학습 데이터는 기기 또는 비공개 iCloud 컨테이너에 로컬 저장됩니다.
  • 개인정보 보장: 타겟 광고를 위해 데이터를 사용하지 않으며, 모델 학습이나 이차 사용을 위해 제3자와 데이터를 공유하지 않습니다.

귀하의 권리

  • 앱 내에서 데이터 조회 및 삭제
  • 언제든지 계정 삭제 가능 — 모든 데이터가 영구 삭제됩니다
  • iCloud 동기화를 비활성화하여 데이터를 기기에만 보관
